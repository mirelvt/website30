Overview of frontend tools and frameworks in the wild.

Februari 21, 2020

Reason why I made this list is because I was curious how much tools and frameworks are available. And as you can see the list is quite overwhelming. And proves in my opinion that as front end developer you first need to have a solid knowledge of HTML, CSS, SVG and Javascript before jumping into a framework (if you really need one).

Here's the list, it is ordered alphabetically. (While writing this post, new ones are published and existing ones might be deprecated.)
